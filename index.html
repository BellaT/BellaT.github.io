Shindig

Shindig is an app created for music lovers everywhere. Here they can share the music they love with their friends and invite them to contribute to the party. Shindig enables a user to make a playlist using YouTube videos and invite friends to add to it from anywhere. All they have to do is log in to the app and add a track and it will be automatically added to the playlist queue (implemented by using sockets). This way everyone can be the DJ. Built with AngularJS, Node, Mongoose, Grunt, SASS, Sockets.

ZNN

ZNN was a group project where we were given the task of making a Node app using an API, and we thought what better use of an API than to create a app explaining how to survive a zombie apocalypse. Google Maps and Google Places were the main APIs and we even included a heatmap to show the zombie infestation. ZNN was built with JavaScript, jQuery, Express, Node, Mongoose.

Pitch

A festival finding app. Building on my love of music and festivals, I made an app where a user can search by either country, genre, month or festival to find one they would like to know more about or purchase tickets for. With this project I was able to spend more time on the front-end and experiment with styling tools, such as Parallax.js and Foundation. This app was built using Ruby on Rails, Bcrypt, Ransack, Carrierwave.

Mastermind

For our first assignment at GA we were asked to build a vanilla JavaScript game, and I chose to make a version of Mastermind. Here the player has to try and beat the computer by guessing a collection of colours and it's order with only 10 attempts. Although this is a simpler app built with JavaScript, jQuery, HTML & CSS...this game is very addictive.
